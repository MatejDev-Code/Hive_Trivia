# The Hive Trivia
Trivia Crack inspired game - 4 scenes, a login, main menu, a game page, and a “live” data/stats page. 
The main menu contains the game start button. 
The game page shuffles through categories of questions and gives you random questions until you fail 3 times or exit the game by pushing exit button. 
The stats page shows the database keeping track of your scores by category(your best and worst).
